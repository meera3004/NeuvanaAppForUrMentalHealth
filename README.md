# NeuvanaAppForUrMentalHealth
Nuevana is a comprehensive self-care app offering breathing exercises, soothing soundscapes, and journaling tools. Features mood tracking, progress analytics, and personalized recommendations to support your mental health journey. Available in light/dark mode with multilingual support.
